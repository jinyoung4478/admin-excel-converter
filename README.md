# 간식서비스 엑셀 변환

현대차 간식서비스 원본 엑셀 파일을 시스템 입력용 형식으로 변환하는 웹 앱입니다.

## 사용 방법

1. 웹사이트 접속
2. 원본 엑셀 파일 선택
3. 매핑 테이블 파일 선택
4. [변환하기] 클릭
5. 결과 파일 자동 다운로드

## 결과 파일 구성

| 시트 | 내용 |
|------|------|
| 데이터 | 일자, 코드, 사업장명, 품목명, Box 입수 |
| 검증 | 원본 데이터와 비교 결과 |
| 매장별 상세 | 매장/일자별 Box 합계 |
| 매핑실패 | 매핑되지 않은 매장 목록 |

## 매핑 테이블

엑셀 파일로 관리하며 아래 컬럼이 필요합니다:

| 컬럼 | 설명 |
|------|------|
| 코드 | 시스템 사업장 코드 |
| 원본 사업장명 | 원본 엑셀의 매장명 |
| 사업장명 | 시스템 등록 사업장명 |

## 배포

GitHub Pages로 자동 배포됩니다.
- `main` 브랜치에 푸시하면 자동 배포
